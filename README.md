# 🌍 EcoTrace — Carbon Footprint Awareness Platform

> **PromptWars Challenge 3: EcoTrace**

EcoTrace is a comprehensive, multi-page Carbon Footprint Awareness Platform built with HTML5, CSS3, and vanilla JavaScript. It empowers users to measure, understand, and reduce their carbon footprint through interactive tools, AI-powered insights, and gamified challenges.

🔗 **Live Demo:** [https://eco-tracee.netlify.app](https://eco-tracee.netlify.app)

---

## ✨ Features

### 🧮 Carbon Footprint Calculator
- Multi-step calculator covering **Transport, Food, Energy, and Shopping**
- Real-time CO₂ score updates as you adjust inputs
- Comparison against India average, world average, and tree offset equivalents
- Emission factor search with 50+ categories
- Save results to Firebase or local storage

### 📊 Dashboard
- Interactive donut chart for category breakdown (Transport, Food, Energy, Shopping)
- Monthly trend line chart tracking footprint over time
- Score cards for daily, weekly, and monthly estimates
- City average comparison with progress bar
- Activity log of recent eco actions

### 🤖 AI EcoBot (Gemini-Powered Chatbot)
- Floating chat widget available on every page
- Answers sustainability, climate, and eco questions using Google Gemini AI
- 6 suggested quick questions to get started
- Conversation history with context awareness
- Smart fallback responses when API is unavailable
- Glassmorphism design with smooth animations

### 💡 AI-Powered Tips
- Personalized eco tips generated by Google Gemini
- Category filtering (Transport, Food, Energy, Shopping)
- "Mark as Done" to earn Green Points
- Daily refresh with caching

### 🗺️ Green Action Map
- Google Maps integration for finding eco-friendly locations
- Categories: EV Charging, Recycling Centers, Tree Events, Organic Markets
- Geolocation support with place search
- Fallback demo map when API keys aren't configured

### 🏆 Challenges & Gamification
- Weekly eco challenges with point rewards
- Badge progression system (Seedling → Eco Starter → Tree Guardian → Earth Defender)
- Public leaderboard from Firestore
- Streak tracking for consistent action

### 📰 Awareness Feed
- 31 curated global environmental articles
- Topic filtering (Climate Change, Renewable Energy, Sustainability Tips, World News)
- "Read & Earn" points system
- Random shuffle on every page load for diversity

### 🌐 Country-wise CO₂ Emissions
- Interactive table showing top 15 emitting countries
- Year tabs from **2020 to 2025**
- Rank badges, flag emojis, and relative bar charts
- Share percentages for each country

### 👤 Profile & Authentication
- Firebase Authentication (Google OAuth + Email/Password)
- User profile with stats, history, and badge display
- Edit profile (name, photo URL)
- Account deletion with confirmation guard
- Auth-aware UI across all pages

## 🔒 Security

- **Content Security Policy (CSP)** on all pages via `<meta>` tags and Netlify headers
- **Firestore Security Rules** with:
  - User ownership verification (`owns(userId)`)
  - Field-level validation (`safeUserFields`, `safeFootprintFields`)
  - Point change caps (+200 max per update)
  - New accounts start at 0 points
  - Immutable footprints and activities (no updates allowed)
- **Security headers**: `X-Content-Type-Options`, `X-Frame-Options`, `Referrer-Policy`, `Permissions-Policy`
- **Auth guards** on protected pages (dashboard, profile)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ (for testing only — the app runs without Node)
- A Firebase project (optional — works in demo mode without it)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/deveshkushwaha/PromptWars-Challenge-3-EcoTrace.git
   cd PromptWars-Challenge-3-EcoTrace
   ```

2. **Install test dependencies**
   ```bash
   npm install
   ```

3. **Run tests**
   ```bash
   npm test
   ```

4. **Open locally**
   Simply open `index.html` in a browser, or use a local server:
   ```bash
   npx serve .
   ```

### Firebase Configuration
Update `js/config.js` with your Firebase project credentials:
```javascript
firebase: {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  appId: "YOUR_APP_ID",
}
```

### Deploy Firestore Rules
```bash
npx firebase-tools deploy --only firestore:rules --project YOUR_PROJECT_ID
```
